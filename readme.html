Quick Start Guide

Initial Setup

IMPORTANT: If your Petz .exe is configured to "Run as Administrator", PetzEL must also be run as administrator! Otherwise PetzEL will not be able to automatically start the game.

New Installation

  • Download PetzEZL zip and extract the contents to any non-Program Files directory, such as your "Documents" folder (tip: you can always find the latest release on the github page)
  • (Optional, but recommended) Download the baseGameBackups zip, and copy the baseGameBackups folder into the same directory as the PetzEZLoader.exe file. This will enable you to use the Base Game File Fallback feature, which will ensure that your game never launches without necessary files.
  • Upon first launch, the program will automatically open the settings menu. Select the "Game Source" button, and navigate to the location of your Petz/Babyz exe file
  • Once the Petz exe file is set, you can proceed to Content Profile Setup or Auto Swapping Setup, depending on which features you wish to use
  • Please read the General Setup Info section for more tips about how to use the EZLoader

Updating a Previous Version

  • Download the newest version (tip: you can always find the latest release on the github page) and extract the contents into the same directory as the previous installation, overwriting any files with the same name
  • (Optional, but recommended) Download the baseGameBackups zip, and copy the baseGameBackups folder into the same directory as the PetzEZLoader.exe file. This will enable you to use the Base Game File Fallback feature, which will ensure that your game never launches without necessary files.
  • Launch the program. Depending on the version you are upgrading from, the PetzEZL program will attempt to automatically migrate your Content Profiles into the new format, however, you will have to manually copy the SeasonalArea files. Once finished, feel free to delete the old Files folder.
  • The PetzEZLoader will also attempt to copy your settings, but a lot new features have arrived. Check Content Profile Setup or Auto Swapping Setup for more information.
  • Please read the General Setup Info section for more tips about how to use the EZLoader

General Setup Info

  • Once you have completed this initial setup, PetzEZL will not open the settings menu unless the "Always Open Settings" option is checked. However, you can always open the settings menu again by double clicking the "modify_setting.bat" file included in the installation directory.
  • In order for the PetzEZL features to work properly, it is recommended that you alway use PetzEZL to launch your game. If you'd like, you can create a shortcut to PetzEZL and rename it to whichever game version you're running, and change the icon to one of the included Petz icon files
  • If you wish to use PetzEZL with more than one P.F. Magic game, you can have multiple installations on one computer, all configured to run different games. Just make sure the files being loaded match the game version you're running! (Note: it is recommended to download a fresh copy of the program for each game, rather than copying a previously installed version, as the EZLoader will enable or disable features depending on the selected game)

Content Profile Setup

Content Profiles (previously known as Bulk Loading) allows you to easily add and remove folders full of content from your game with a single click. The flexibility of content profiles means they can be used for almost anything - managing your seasonal toyz and clothes, building sets of your favorite OW files for breeding, swapping in and out WIP hexies, and even more! Additionally, the new Petz Profile feature allows you to create profiles containing .pet files, along with their textures, palettes, and breedfiles, and even their favorite toyz and clothes, all while maintaining their memories.

Getting Started

  • First click "Enable Content Profiles" to enable the feature.
  • Next, go to the Create New Profile box. Enter the name you wish to call your new profile, and check the types of content you want to include in that profile. If don't know for sure yet - don't worry. You can add or delete profile subfolders at any time. If the content type you want to include is grayed out, check that the "Disable" option in the the Loading Override box is unchecked (Note: Some types of content may be disabled completely depending on the game version you load. If so, that means that type of content isn't compatible with that game). When finished, click the "Create Profile" button, and the program will automatically generate the profile folders for you.
  • Note 1: When using Adopted Petz profiles, it is recommended that each pet only be included in one profile. Otherwise, it is possible that the latest version of the pet might be overwritten
  • Note 2: Some kinds of content - Carrying Cases, Mice, and AC Sprite - require an edited PetzRes.dll file to be used. See the FAQ section for more info
  • Once you've created your first profile, navigate to your EZLoader directory. Enter the "Filez" folder, then "ContentProfiles". You should see a folder with the same name as the profile you just created. Copy the files you wish to be included in the profile into the same folder you would put them in the game (with the exception of Cases, ACSprites, and Mice, which should be copied into their respective folders).
  • Go back to the EZLoader. New profiles should automatically get added to the Include Box, though if it's not there, you can click the "Refresh Folders" button to have the EZLoader scan for folder changes and update the UI. All the profiles in the Include Box will be copied into your game the next time you click "Save and Start Petz" (or start the EZLoader without opening the settings menu). Try it now, and confirm that the game content you added to your new profile is now in your game.
  • Once you've confirmed this, close the game and open the EZLoader settings again. Now, select a profile in the Include Box and click the right arrow button to move it to the Remove Box. Start the game again, and the content in that profile will be gone!
  • If you wish to delete a profile, please either use the included delete feature, or ensure the profile is in the "Remove" section before you delete it. Otherwise, you will have to remove the files from your game manually
  • That covers the basics of the Content Profile system. For more advance tips, check out the full feature list.

Auto Swapping Setup

The Petz EZLoader isn't just a content management system. It actually adds new features to the game! By checking your computer's date and time and automatically swapping out certain types of game content such as playscenes before loading the game, it creates the illusion that Petz 3/4 and Babyz have real time, seasons, and weather.

Note: Carrying Cases, Mice, and AC Sprite swapping require an edited PetzRes.dll file to be used. See the FAQ section for more info.

Getting Started

  • First, click "Enable Auto Swapping" to enable the feature.
  • The Auto Swapping feature allows for 3 different types of swapping - Seasonal, Day/Night, and Weather. It also allows the swapping of 3 different kinds of content - Playscenes, Carrying Cases, and Mice. These can all be mixed and matched to customize the system to work how you want.
  • To setup auto swapping, simply navigate to your EZLoader folder, then open the "Filez" folder. Within, you'll see 4 "Seasonal" folders, each with a number of subfolders - first season, then time, then weather. Simply copy the OW content you want to be loaded for a particular combo into that folder.
  • Each type of swapping also has a default folder that you can set. If you don't enable a type of swapping for a particular piece of content, that's the folder that will be loaded instead. For example, if you don't want to use weather with playscenes, and the default weather is set to "clear", the EZLoader will always load the content in the "clear" subfolder. So if it's day time in the winter, it will load winter/day/clear. And at night, it will load winter/night/clear.
  • If a particular folder doesn't have an OW file for that combination, the behavior will change based on if you have Base Game Fallback enabled. If you do, it will simply load the original game file. If not, it will load whichever file was last used in the game.
  • Carrying Cases have a special option - 24 hour mode. This is designed to replicate the PII carrying case, which had new art for the interior of the case for each hour of the day. To use this option, copy your set of cases into the 24hour case folder in the SeasonalCase directory. Clicking this option disables the weather and seasons because I don't think anyone is crazy enough to make a full set of art for every combination. If someone proves me wrong and does it, then I'll enable this feature.
  • This covers the basics of Auto Swapping. Each type of swapping also has some additional settings you can tweak it to match your personal preferences. Most of these should be self explanatory, but if you need more help, check out the full features section.

Feature Glossary

Here you can find an in-depth explanation of all the various features of the EZLoader, for easy reference.

Main Buttons

Buttons that are visible on all tabs of the program

  • Move Files - saves your settings, then moves the game files based on your current settings, but doesn't start the game
  • Save - saves your settings, but doesn't start the game
  • Save and Start Petz - saves your settings, moves the files, then starts the game (or opens the Content Profile Manager if that option is checked)
  • Help - opens this Help Guide

General Setting

Setting that relate to both main features of the EZLoader program, or the program in general

General

  • Game Source - The location of Petz or Babyz program that's being modified
  • Edit Source - allows you change the above info
  • Always Open Settings - when checked, opens the main settings window each time you launch the program
  • Always Open Content Profile Manager - when checked, opens the Content Profile Manager window each time you launch the program. For more info, see the Content Profile Manager section

Advanced Settings

Settings you probably won't need, unless you're a power user or run into a problem

  • Custom Load Order - changes the order that the two main EZLoader features are applied. Useful when dealing with OWs. In either case, the base game fallback applies first.
  • Disable Base Game Fallback - when checked, disables the base game fallback features. Useful if you for some reason don't want all the original game content in your game at all times (in which case you'll have to manage replacing OW data yourself by creating profiles). Checking and unchecking it will refresh the baseGameBackup folder in content profiles (useful if you accidentally change that data), and it will be automatically disabled if it can't find the baseGameBackups folder when trying to load the backup data.
  • Regenerate Folders - recreates the "Filez" folder structure with all the Seasonal subfolders
  • Reset Settings - resets all the setting to defaults
  • Create debug.txt - creates a "debug.txt" file in all the Seasonal subfolders with the path of that particular file subfolder as the content. Useful for debugging which folder is being loaded.
  • Delete debug.txt - deletes the above generated "debug.txt" files in all the Seasonal subfolders. If any files are left in the game folders, you will need to delete them manually

Content Profile Settings

Settings on this tab related to the Content Profiles feature, which allows you to move entire folders of game content in and out of your game with a single click

General

  • Enable Content Profiles - toggles the entire feature
  • Sort Remove - sorts the folders in the "Remove" box alphabetically
  • Refresh Folders - checks if any new folders have been created or removed in the Filez\ContentProfiles directory, and updates the UI accordingly
  • Include Box - the contents of the folders in this box will be copied into your game the next time you hit "Move Files" or start the game (with the exception of Adopted Petz/Babyz, which will only be copied if they aren't already there, in order to maintain the latest version). You can use the arrows to the left of the box to rearrange the order these folders are loaded - folders on top will be loaded first. This can be useful for managing OWs.
  • Remove Box - the contents of the folders in this box will be deleted from your game (but not from your Filez folder) the next time you hit "Move Files" or start the game (except for Adopted Petz/Babyz will be copied back into the folder in order to maintain the latest version).
  • Arrow Buttons - the up and down arrows allow you to rearrange the contents of the Include Box. The left and right arrows allow you to move content between the two boxes
  • Filter Box - the include and remove boxes will filter their content based on what you type in this box. Useful for finding a specific folder.

Loading Override Settings

Allows you to enable or disable loading specific types of content. For performance reasons, it is recommended to only enable loading for the types of content you intend to swap. Note that to swap Carrying Case, Mice, and AC Sprites, you will need to have an edited Petz Rez.dll file (for more info click here). Also, these setting only apply to content within the ContentProfiles folders. If you have swapping enabled for content in the AutoSwapping tab, it will continued to be swapped as normal

  • Disable Loading Resources - disables loading all content in the Petz/Resource subfolders
  • Disable Loading Adopted Petz - disables swapping Petz/Babyz in your Adopted Petz/Babyz folder
  • Disable Carrying Cases - disables loading custom carrying cases
  • Disable Mice - disables loading alternate mice lnz files
  • Disable Loading Resources - disables loading alternate Adoption Center sprites

Create New Profile

This section allows you to easily create a new content profile folder with whichever you want. Note that you can still create profile folders manually, as well as add or delete subfolders at any time, including subfolders not listed as options. This is just a shortcut to enable you to quickly and easily create new profiles with the most common folder types.

  • Name - the name you wish to give the new profile. If you type in the name of an existing profile, it will instead generate any additional subfolders you check for that folder (but not delete the unchecked ones, to avoid accidentally deleting content)
  • Various Checkboxes - allows you to pick which subfolders you want to be generated for the new profile
  • Clear Selection - unchecks all the checkboxes
  • Create Profile - creates the profile

Delete Profile

Enables you to delete a profile from both your game, and your Content Profile folder. Only use this if you've already backed up any content you wish to keep. To delete a profile, simply select it from the drop down and click delete. You can also start typing in the drop down box to filter the selection. If you want to delete a profile without using the feature, please make sure to move it to the "Remove" section and click "Move Files" first. Otherwise you will have to remove those files from your game manually.

Content Profile Manager

The Content Profile Manager is a new windows that can be enabled, giving you a way to view and manage your Content Profiles each time you start you game, with even more room and filter options. The functionality of this window is mostly the same as the content profiles tab, with the exception of the "Category Filters" section. This section will allow you to filter profiles based of off what type of content they contain. Note that these filters will show any profiles that contain a folder for that type of content, regardless if that folder is empty.

Auto Swapping Settings

Settings on this tab relate to the Auto Swapping feature, which allows you to setup certain types of game content to automatically swap out based on the time and date of your computer's clock and (optionally) weather data based on the entered coordinates. Note that to swap Carrying Case, Mice, and AC Sprites, you will need to have an edited Petz Rez.dll file (for more info click here).

General

  • Enable Auto Swapping - toggles the entire feature
  • Default Season/Time/Weather - selects the folder that will be loaded if a particular type of content is not enabled for that type of swapping
  • Enable Playscene Swapping - enables playscene swapping for that type of auto swapping. Works with both base game and custom playscenes
  • Enable ACSprite Swapping - enable the swapping of externalized Adoption Center sprites, sometimes used with custom Adoption Center playscenes
  • Enable Case Swapping - enables the swapping of externalized Carrying Case sprites
  • Enable Mice Swapping - enables the swapping of externalized Mice lnz

Seasonal Options

  • Season Start Date - Selects the day of the year that the season changes. Can be tweaked to reflect how the seasons typically change in your area.

Day/Night Options

  • Sunrise/Sunset Times - Sets the time of day in which night switches into day, and day into night. Once again, this can be tweaked to match your local area
  • Enable unique sunrise/sunset swaps - allows you to use special playscenes for the hours in which sunrise and sunsets occur
  • Enable 24 hour case mode - enables a special 24 hour mode to allow you to use unique carrying case art for each hour of the day. Designed specifically to replicate the PII case.

Weather Options

  • Weather rotation style - Allows you to pick between 3 weather rotation styles
    • Random on load - picks a random weather each time the game loads
    • Random per day - picks a random weather the first time the game is loaded for the day, and keeps that weather for the whole day
    • Location based - uses the provided longitude and latitude info to check the current weather using the free Open-Meteo api. Requires internet access.
  • Latitude/Longitude - the data used to lookup weather for a particular location. You can find these for your location with a simple google search.

FAQ

  • Where can I find Auto Swapping compatible scenes?
  • There are a number of people around the community who have created scenes specifically for use with the EZLoader. Here are just a few:
    • Butterfly Fieldz
    • Phantasmagoria

  • My Carrying Cases, Mice, and/or AC Sprites aren't swapping? What do I do?
  • Using these swapping features requires use of an edited Petz Rez.dll. For convenience, I have provided an edited Petz 4 Rez.dll for download on the EZLoader page. However, if you wish to use the EZLoader with a different version of Petz, or only want to externalize some of the content, you can edit the dll yourself. Simply open the dll in LNZPro, and right click > delete all the files related to the type of content you wish to externalize. For a list of these files, check the baseGameBackups folders.

  • I downloaded a custom Carrying Case, Mouse, or Adoption Center scene, but the content is only offered bundled in the Petz Rez.dll. How do I use it with the EZLoader?
  • As the popularity of the EZLoader grows, I suspect most Petz content creators will begin to offer external versions of their custom cases, mice, and scenes. However, you may still find content only offered as a Rez.dll. To externalize it, simply open up the edited Rez.dll in LNZPro and export the relevant files yourself. To export a file, right click it, then click "save to file". Type in the name of that file exactly, including the extension. Be careful not to mix up the file names! For a list of which files need to be saved for each type of custom content, check the baseGameBackups folders.
